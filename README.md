# Klage-permittering-refusjon 

Klage på vedtak for refusjon ved permittering

## Komme i gang

### Avhengigheter
For å kjøre lokalt kreves det at man har node og yarn installert

### Kjør lokalt
  * kjør `yarn install` 
  * kjør `yarn start`, for å bygge og kjøre frontend lokalt
